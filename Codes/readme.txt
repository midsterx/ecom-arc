1. Always run the code in sudo mode - Python 2
	Command: python firewall_scapy+nfq.py
2. Dependencies: Scapy and NetfilterQueue
3. The Configuration File pertains to three modes for blocking traffic:
	a. true
	b. false
	c. test - which corresponds to blocking traffic from localhost alone (127.0.0.1)
4. The rules of the config file can be extended as per user requirements.
